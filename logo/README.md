# Logo — SVGs

Símbolo + variações de wordmark, tudo em SVG vetorial.

| Arquivo | Uso |
|---|---|
| `trilho-mark.svg` | Símbolo isolado, fundo claro |
| `trilho-mark-light.svg` | Símbolo isolado, fundo escuro |
| `trilho-lockup-horizontal.svg` | Logo horizontal "Trilho." (uso primário) |
| `trilho-lockup-horizontal-light.svg` | Logo horizontal, fundo escuro |
| `trilho-lockup-stacked.svg` | Logo empilhado (assinaturas, papelaria) |
| `trilho-cred-lockup.svg` | Lockup do produto com tagline mono |
| `favicon.svg` | Favicon 64×64 com fundo marinho arredondado |

**Cores**: marinho `#0A1538`, esmeralda `#00D49A`, off `#FAFAF7`
**Fonte do wordmark**: Geist Medium (fallback Inter). O texto está como `<text>` — converta para outline (`<path>`) se for embedar onde Geist não estiver disponível.
